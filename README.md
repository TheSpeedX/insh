insh  (insulter shell)
=========

Insults You Whenever You Get An Error !!!

Installation
=========

`pip install insh`

or install from source

`pip install git+https://github.com/TheSpeedX/insh.git`

## Usage

```
Usage: insh <commands>
or:  insh   #To start a shell

insh help   #For help

```
